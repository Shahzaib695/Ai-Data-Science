# AI and Data Science Study Repository

This repository serves as a structured collection of resources, programming concepts, and projects focused on Python programming, data analysis, and machine learning. It contains both personal study materials and structured coursework.

## Repository Structure

The codebase is divided into two primary sections:

### 1. Python Fundamentals
This folder contains independent study notebooks, custom scripts, and exercises covering foundational Python concepts up to intermediate programming levels.
* **Topics**: Variables, string manipulation, data structures (lists, tuples, dictionaries, sets), conditionals, loops, functions, file input/output, and advanced Python concepts (lambdas, list comprehensions, decorators, args/kwargs).
* **Object-Oriented Programming (OOP)**: Revision files and deep dives into the pillars of OOP (inheritance, polymorphism, encapsulation, abstraction) and dunder methods.
* **Projects**: Sample implementation projects, including terminal games like Snake, Water, Gun.

### 2. SMIT Class Work
This folder contains organized lectures and coursework from the SMIT AI and Data Science program.
* **01_Python_Basics**: Structured lecture notebooks covering Python syntax and OOP implementations.
* **02_NumPy**: Numerical analysis and array manipulation.
* **03_Pandas**: Data structures, CSV and Excel file read/writes, filtering, and data cleaning.
* **Assignments**: Homework notebooks submitted during the course (kept at the root of the class folder to preserve submission links).

## Installation and Setup

### Prerequisites
* Python 3.x
* Jupyter Notebook or JupyterLab

### Library Dependencies
Install the required libraries for the data science modules:
```bash
pip install numpy pandas pillow openpyxl
```

### Running the Notebooks
To explore the notebooks locally:
1. Clone this repository:
   ```bash
   git clone https://github.com/Shahzaib695/Ai-Data-Science.git
   ```
2. Navigate to the directory:
   ```bash
   cd Ai-Data-Science
   ```
3. Launch Jupyter:
   ```bash
   jupyter notebook
   ```
