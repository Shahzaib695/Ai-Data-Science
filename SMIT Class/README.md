# SMIT Class Work

This directory contains the coursework, lecture notebooks, and assignments from the SMIT AI and Data Science course. The materials are organized by topic to maintain a clean structure while keeping the assignment folders at the root level to preserve external links.

## Directory Structure

* **01_Python_Basics**: Core Python syntax, loops, functions, file handling, and Object-Oriented Programming (OOP) concepts (Classes 1 to 17).
* **02_NumPy**: Multi-dimensional arrays, mathematical operations, and image processing basics using NumPy (Classes 17 to 18).
* **03_Pandas**: Data manipulation, filtering, aggregation, and file handling (CSV/Excel datasets) using Pandas (Classes 19 to 23).
* **Assignment1 to Assignment5**: Practical notebooks containing homework and assignments, kept at the root to maintain link integrity for submissions.
